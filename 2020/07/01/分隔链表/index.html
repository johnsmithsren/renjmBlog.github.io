LeetCode - 分隔链表题解

发表于2020-07-01|更新于2025-09-21|算法

题目描述

给定一个链表和一个特定值 x，对链表进行分隔，使得所有小于 x 的节点都在大于或等于 x 的节点之前。需要保持节点的相对顺序。

解题思路

双指针法

代码分析

  1. 创建两个虚拟头节点,分别存储小于x和大于等于x的节点
  2. 遍历原链表,根据节点值分配到对应链表
  3. 最后将两个链表连接起来
  4. 注意处理尾节点的next指针

复杂度分析

  • 时间复杂度: O(n)
  • 空间复杂度: O(1)

注意事项

  1. 需要断开大值链表的尾部指针
  2. 保持节点的相对顺序不变
  3. 处理好链表为空的边界情况
