格式说明

常用的格式化参数含义:

  • %h: 提交的简短哈希值
  • %d: 引用名称(分支、标签等)
  • %s: 提交信息
  • %cr: 相对时间
  • %an: 作者名字
  • %Cred/%Creset: 颜色控制

实用技巧

  1. 使用 --oneline 可以获得简洁的输出格式
  2. --graph 选项可以可视化显示分支合并历史
  3. --pretty=format 支持自定义输出格式
  4. 合理使用别名可以简化常用命令
